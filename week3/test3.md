7. Які значення приймуть a та b після виконання main
```java
class Test
{
    int a, b;
    Test()
    {
        a = 10;
        b = 20;
        }
}

class Runner
{
    public static void main(String[] args)
    {
        Test obj1 = new Test();
        Test obj2 = obj1;
        
        obj1.a += 1;
        obj1.b += 1;

        obj2.a += 1;
        obj2.b += 1;
    }
}
```
- [ ] 11, 21
- [X] 12, 22
- [ ] 10, 20
- [ ] 2, 2
